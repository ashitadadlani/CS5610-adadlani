AngularJS

In this week, I have tried basic functionalities in AngularJS.The experiments use bootsrap as well.
I have tried the following:

  • Filters: I have created a form and used uppercase and currency filters, and pipe(|) character
  • Tables: In this, I have created a table with pre-defined data and used ng-repeat directive to search in the table
  • Adding/Removing from table: Here I have tried dynamically adding and removing from the table with pre-existing data.
  • Validations: In this experiment, I have tried validating several input fields of a form
  • Event Handling: Here, I used event handlers for DOM elements
