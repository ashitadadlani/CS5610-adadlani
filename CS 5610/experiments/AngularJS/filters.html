AngularJS Forms


About the Experiment

In this experiment, I am exploring the filter functions in Angular JS. The data in input fields is defaulted using ($scope). We can modify the data entered by the users using filter functions. The data entered can be clubbed together using pipe(|) character.
The input value can be converted to uppercase using 'uppercase' filter.
Multiplication of price per item and number of items is done using 'currency' filter.

Code Snippet

The HTML Code below shows the use of filter functions 'uppercase' and 'currency' followed by pipe character.

            <div ng-app="" ng-controller="costController">
                <div class="row">
                    <div class="col-md-6">
                        <b>First Name: </b><input type="text" ng-model="lastName" ><br><br>
                        <b>Enter book name: </b><input type="text" ng-model="book"><br /><br />
                        <b>Enter quantity: </b><input type="number" ng-model="quantity"><br /><br />
                        <b>Price per item ($):</b> <input type="number" ng-model="price"><br /><br />

                        Full Name: {{firstName + " " + lastName}}<br />
                        Book ordered: {{book | uppercase}}<br />
                        <p><b>Total Cost:</b> {{ (quantity * price) | currency }}</p>
                        </div>
                    </div>
                </div>
            </div>

The Javascript code below shows the coltroller function to default the input field with some value.

            function costController($scope) {
               $scope.quantity = 1;
               $scope.price = 9.99;
               $scope.book = "The Alchemist";
               $scope.firstName = "Ashita",
               $scope.lastName = "Dadlani"
            }
