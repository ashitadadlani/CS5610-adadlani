Event Handling

About the Experiment

In this experiment, I have tried two event handlers for DOM elements.
In the first one,that is, when the user checks the check box 'disable', the button 'Click me' gets disabled.In the second, I have used ng-click directive with the button to define what needs to be done on click.

Code Snippet

The HTML Code below shows the use of ng-click, ng-model and ng-disabled directive.
The ng-click directive defines the expressions to be evaluated when the button is clicked.
ng-model directive is to link existing html elements with a value in scope.
ng-disabled directive disables the button. 

                <p> <button class="btn btn-primary" ng-click="count = count + 1"> Click Me! </button> : {{ count }} </p>
                <input type="checkbox" ng-model="enableDisableButton">Disable Button
                <button class="btn btn-primary" ng-disabled="enableDisableButton" ng-click="click()">Click Me!</button>

The Javascript code below initializes the count variable to zero and throws a pop-up when button is enabled and clicked.

                app.controller("myController", function ($scope) {
                    $scope.count = 0;
                    $scope.click = function () {
                         alert("button clicked");
               }}
