Experiment Details

In the experiment, we create a navigaton header having several navigation links on it. These belong to the class <nav> and to create a fixed header, we use the class "nav-header". Then we have implemented <jumbotron> and inseerted an image, a paragraph and a button to it. The search symbol is drawn on the search button using ""glyphicon glyphicon-search" class.
