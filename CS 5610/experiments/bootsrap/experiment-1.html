Experiment Details

The .table class adds basic styling (light padding and only horizontal dividers) to a table.Bootsrap grid system contains for different classes: xs(for phones), sm(for tablets), md(for desktops) and lg(for larger desktops).We define a row in a table using <tr> tag, a heading using <th> tab and the content of an element using <td> tab.
