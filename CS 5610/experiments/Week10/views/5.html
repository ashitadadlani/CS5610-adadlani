Carousel and Tabset


Experiment Details

In this experiment, I have demonstrated a carousel which changes images at a constant interval and keeps on repeating the images after the array is fully traversed. I will be using this carousel to show the images of car that the user has selected. I have also tried working and creating a tabset which display it's respective information when clicked. The Dynamic tab allows you to write your own comment. I will use different tabs in tabset to display information about the car such as it's price, features, reviews and the dealers in the nearby location.

Code Snippet

HTML code for Carousel:

            <div id="myCarousel" class="carousel slide" data-ride="carousel" ng-show="media">
               <div class="carousel-inner" role="listbox">
                  <carousel interval="myInterval">
                     <slide ng-repeat="m in media">
                        <img src={{m}} height="100%" width="100%" />
                     </slide>
                  </carousel>
               </div>
            </div>

HTML Code for Tabset

<tabset justified="true">
    <tab heading="Heading1">
       <p>Some contents in Heading1</p>
    </tab>
    <tab heading="Heading2">
       <p>Some content in Heading 2.</p>
    </tab>
    <tab heading="Add new Content">
       <button class="btn btn-primary" ng-click="add()">Add Review</button>
       <div ng-show="newContentToAdd">
          <textarea id="text" rows="4" cols="75" ng-model="content.text" placeholder="Add new content"></textarea>
       <div>
       <button id="contentSubmit" class="btn btn-info" ng-show="content.text" ng-click="submitContent(content)"> Submit</button>
       <button id="contentSubmit" class="btn btn-info" ng-click="cancelContent()"> Cancel</button>
    </tab>
</tabset>
