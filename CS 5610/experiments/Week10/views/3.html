Creating Dropdowns


About the Experiment

In this experiment, I have created a dropdown. When we click on 'Search by Makes', all the car makes get fetched. Once we select the car make, Car model button gets shown and similarly, car year and styles button.

Code Snippet

<li class="dropdown">
    <a data-toggle="dropdown" href="#home" ng-click="fetchMakes()">Search by Makes<span class="caret"></span></a>
        <ul class="dropdown-menu">
            <li class="dropdown-submenu" ng-repeat="make in makes">
                <a tabindex="-1" class="dropdown-toggle" data-toggle="dropdown" ng-click="fetchModels(make.name)">{{make.name}}</a>
            </li>
        </ul>
    </li>
<li class="dropdown" ng-show="makeName">
<a data-toggle="dropdown" href="#home">Models<span class="caret" ></span></a>
    <ul class="dropdown-menu">
        <li class="dropdown-submenu" ng-repeat="model in models">
            <a tabindex="-1" class="dropdown-toggle" data-toggle="dropdown" ng-click="fetchYears(model.name, makeName)">{{model.name}}</a>
        </li>
    </ul>
</li>
<li class="dropdown" ng-show="modelName">
    <a data-toggle="dropdown" href="#home">Year<span class="caret" ></span></a>
        <ul class="dropdown-menu">
            <li class="dropdown-submenu" ng-repeat="year in years">
                <a tabindex="-1" class="dropdown-toggle" data-toggle="dropdown">{{year.year}}</a>
            </li>
       </ul>
</li>
