jQuery Interactions

1. Draggable

The image can be dragged and dropped anywhere on this screen.


Experiment Details

The dragging is handled by the function .draggable().

Code Snippet

            $(function () {

            $("#draggable").draggable();

            });

2. Drag and Drop, and Sort

When we drag any element inside the box, all the elements get rearranged. If you try to move
the element up, the elements move to the right from the dropped position, and when tried to
drag down, they move left.

Experiment Details
