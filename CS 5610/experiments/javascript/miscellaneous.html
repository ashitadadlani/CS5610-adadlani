2. Focus

Try entering the name and email below. As you click on Name or Email input box, the background color changes from white to grey.Also, when input field is clicked, you can see an instruction displayed which slowly disappears.


Name: You name should be in block letters!!
Email: Enter complete email address.

Experiment Details:

Both of these actions are performed by focus() function. To fade out the instruction, we use .fadeOut() functin.

Code Snippet:

                
                    $("#input_focus_name").focus(function () {

                    $(this).css("background-color", "#cccccc");

                    $("#name").css("display", "inline").fadeOut(2000);

                    });

                    $("#input_focus_email").focus(function () {

                    $(this).css("background-color", "#ffffff");

                    $("#email").css("display", "inline").fadeOut(2000);

                    });
