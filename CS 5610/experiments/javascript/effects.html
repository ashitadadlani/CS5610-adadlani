jQuery Effects

1. Show/Hide



Click on Show/Hide button to see the effect


Experiment Details

Show/hide is implemented using .toggle() function on the button.

Code Snippet

            $("#show-hide-btn").click(function () {

               $("#show-hide").toggle();

            });

2. Fade In/Fade Out

Click on the button to see Fade In/Fade Out effect




Fade In-Fade Out Effect

Experiment Details

The fadeToggle() function toggles between Fade In in one click and Fade Out on the next click.

Code Snippet

            $("#fadeIn-fadeout-btn").click(function () {

                $("#div1").fadeToggle(2000);

            });

3. Animate

Hover the mouse on the image to see animation effect.



Experiment Details

The image animates(zooms in) when mouse is hovered ober it and zooms out when the mouse is removed.This is done by using .animate() function in which we pass the width and height of the image after zooming in and zooming out.

Code Snippet

            $("#animate-test img").hover(function () {

              $(this).stop().animate({ width: "500px" , height: "500px"});


            }, function () {

              $(this).stop().animate({ width: "200px" , height: "300px"});

            });
