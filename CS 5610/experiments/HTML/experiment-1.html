Experiment Details

<form>element defines HTML forms. The attribute action navigates to the page after clicking submit on the current form.<fieldset> is used to group related elements in a form by drawing a box around the related elements. <legend>defines a caption for the <fieldset> element. <input>specifies an input field where the user can enter data.The attribute 'text' allows to enter text inputs to the field element.'Placeholder' attribute specifies a hint to the value to be entered to the field. 'Required' makes the field mandatory. Type 'Submit' allows to submit the form and move to the following page

Code Snippet

<form action="experiment-2.html">
<fieldset>
<legend> Personal Information </legend>
First Name:<input type="text" name="firstname" maxlength="10">
Email:<input type="email" name="email" placeholder="abc@xyz.com">
Password: <input type="password" name="passwd" required>
</fieldset>
</form>
