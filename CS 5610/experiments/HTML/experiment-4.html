Experiment Details

The <iframe> tag specifies an inline frame used to embed another document within the current HTML document. Src attribute determies the source from where the second document is to enbeded in the current one. Width and height attribute defines the area occupied by the inline frame.

Code Snippet

<iframe align="middle" src="scrolling.html" width="600" height="500" seamless>
