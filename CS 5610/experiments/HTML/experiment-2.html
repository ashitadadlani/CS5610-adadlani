The <datalist>; element specifies a list with pre-defined option for <input> element.The user can type in a new value or select from the existing list of options. <option> is used to define the values for pre-defined options in datalist.

Code Snippet

<datalist id="school">
<option value="Carnegie Mellon University"></option>
<option value="Boston University"></option>
<option value="Northeastern University"></option>
<option value="Stanford University"></option>
<option value="Harvard University"></option>
<option value="UMass Amherst"></option>
</datalist>

 <select>; is used to create a drop-down list.<option> is used to define the values for drop-down list.

Code Snippet:

Field of Study: <select name="study">
<option value=""></option>
<option value="Arts and Sciences">Arts and Sciences</option>
<option value="Commerce">Commerce</option>
<option value="Computer Science">Computer Science</option>
<option value="Information Technology">Information Technology</option>
<option value="Journalism">Journalism</option>
<option value="Other">Other</option>
</select>

 Number attribute defines a text area where a number is to be entered. Min and Max define the minimum and maximum value of this field.

Code Snippet

Duration of course (between 1 and 5): <input type="number" name="quantity" min="1" max="5" value="1">
