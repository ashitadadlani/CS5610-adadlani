Star Rating


Experiment Details

Sometimes when we have to display ratings, displaying it in star format gives a more appealing look to the rating. This experiment uses AngularJS directive to display the rating of something as star representation.

Code Snippet

AngularJs directive

app.directive('rate', function () {
    return {
        restrict: 'A',
        template: '<ul class="rating">' +
        '<li ng-repeat="star in stars" ng-class="star">' +
        '\u2605' +
        '</li>' +
        '</ul>',
        scope: {
            ratingValue: '=',
            max: '='
        },
        link: function (scope, elem, attrs) {
            scope.stars = [];
            for (var i = 0; i < scope.max; i++) {
                scope.stars.push({
                    filled: i < scope.ratingValue
                });
            }
        }
    }
});

This is used in HTML code as follows:

        <rating class="rating" ng-model="rate3" max="5" readonly="true" on-leave="overStar = null"></rating>
